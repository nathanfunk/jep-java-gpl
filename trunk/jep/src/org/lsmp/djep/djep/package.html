Classes to performs differentaion of equations and various utility classes. The main class is {@link DifferentationVisitor DifferentationVisitor} which performs the differentation by repeated applications of the chain, product and quotient rules. Typically the result of differentation should be simplified to remove occurences of reduant expresions (for example diff(x^2.0,x) is 2.0*x^1.0*1.0 which is simplified to 2.0*x).

For each function a rule is used to specify how to differentiate it. For example the differative of sin is specified using 

addDiffRule(new MacroDiffRules(Jep,"sin","cos(x)"))
There are various classes of differentation rule which all implement {@link DiffRulesI DiffRulesI} and have special behavious.

Variables can also be differentiated, and the derivative of y with respect to x will be represented by the {@link PartialDerivative PartialDerivative} dy/dx. To allow this the {@link DVariable DVariable} is used to represent variable and {@link DSymbolTable DSymbolTable} and {@link DVariableFactory DVariableFactory} are used to control the look up of variables and the creation of new variables.
