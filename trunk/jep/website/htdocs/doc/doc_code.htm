JavaDoc Class Documentation

The org.nfunk.jep.JEP Class

This is the only class that needs to be accessed for parsing expressions. It contains all functions needed to add functions and variables. Once an expression has been parsed, the values of the variables can be adjusted and the expression evaluated.

Contents of the org.nfunk.jep.function package

PostFixMathCommandI

The Interface Class for all math commands. It requires the classes which implement it, to contain a run(Stack aStack) function. The parameters for the math command are placed on the stack before calling run, and the result can be popped off afterwards.

PostFixMathCommand

Contains a stack checking function, and a numberOfParameters field. The stack checking function checkStack ensures that the stack passed as argument is not null, and the stack size is at least equal to the number of parameters.

Mathematical Functions and Operators Classes

Each of the classes such as Add, Angle, Sine, or Divide implement PostFixMathCommandI and extend PostFixMathCommand. They all return their result through pushing it on the the top of the stack in the run(Stack aStack) function.
